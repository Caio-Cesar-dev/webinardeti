# Curso com HTML e CSS
## Curso com o professor Matheus Battiste no YouTube
Neste curso foi explorado os recursos fundamentais para trabalhar com HTML, como: tags, atributos, formulários, listas, HTML semântico
E também o que é preciso para criar projetos com CSS: seletores, especifidade, positions, flexbox, display e diversas outras regras.

O resultado foi a criação de uma página de um Webinar com um formulário para incrição.
